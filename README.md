# Python工程师面试题

请用Flask框架搭建一个项目，主要实现功能如下：

+ 用户下单接口

前提条件：

+ 数据库表名: shop
+ 用户表名: users
+ 商品表名: goods
+ 用户帐户余额为100元
+ 商品价格随意

要求：

+ 数据库用Mysql并提供表结构sql
+ 用户下单是用余额下单
+ 项目代码放在project目录下
+ 从[https://github.com/jiangshengjun/QK-INTERVIEW]（https://github.com/jiangshengjun/QK-INTERVIEW)fork代码并以提交Merge Request方式来完成答题
